v0.3 release

Kian · Harshit · Jack · Neel · Mart

We are releasing v3 with a focus on clarity, speed, and simplicity. The interface remains minimal and direct.

Our gym environment now exposes a cleaner action/observation surface and consistent reward semantics. We removed unnecessary layers, tightened the reset/step loop, and standardized output for RL workflows.

We undertook a substantial refactor to reduce coupling and streamline the codebase. Core logic has been simplified, Lua-side responsibilities clarified, and Python orchestration made thinner and faster.

We are now operating under a new organization, reflecting a long-term commitment to a lean, open evaluation environment.
